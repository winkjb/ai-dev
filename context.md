# ai-dev

## Purpose
Root of the ai-dev workspace. Development environment for business automation projects supporting the teams Brad supports: Project Management, Security, and Service Delivery.

## Contents
- `bootstrap.md` — session startup doc; current state, open loops, what's next. Read first, every session.
- `CLAUDE.md` — workspace conventions (folder structure, naming, context.md files, relative paths).
- `principles.md` — the 60/30/10 design principle governing how automation is built here.
- `readme.md` — master index of the workspace.
- `project-management/`, `security/`, `service-delivery/` — one folder per team; each self-contained with its own `context.md`, roles, scripts, and data.

## Notes
See `CLAUDE.md` for the full context.md convention used throughout this repo.
